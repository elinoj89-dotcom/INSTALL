# 🛡️ INSTALLATEUR SMM ELINO

Interface d'installation sécurisée pour les outils SMM de Elino. Ce système vérifie l'accès via une clé unique (HWID) avant de configurer l'environnement.

## 🚀 Installation Rapide

Copiez et collez la commande suivante dans votre terminal **Termux** :

```bash
pkg update && pkg upgrade -y && pkg install git python -y && git clone [https://github.com/elinoj89-dotcom/INSTALL](https://github.com/elinoj89-dotcom/INSTALL) && cd INSTALL && python main.py
